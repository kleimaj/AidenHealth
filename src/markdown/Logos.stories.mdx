# Logo

Logos have the potential to do incredible things for the companies behind them. Logos set the tone for the brand, improve memorability, help users emotionally connect to the brand, and create consistency between all marketing channels. 

<br/>


## Usage 

Use the logo in the site to refer the users to the brand identity, and the company as a whole.

<br/>
<br/>
