# Navbar

Navigation Bars are a section of an interface intended to aid visitors in accessing information, or different parts of the application.

## Usage

Use navigation bars to offer different routes to the user's specific needs.

## Top Navbar

The Top Navigation Bar is used to present instant navigation to the intended pages.

## Mobile Navbar

The Mobile Navigation Bar is used to present pages in an organized fashion with minimal screen real estate.

The `HamburgerIcon` is a styled component within itself, that toggles and animates the dropdown menu component on click!

<Preview>
  <Story name="Mobile Navbar">
      <Navbar modifiers="mobile">
        <h1><a>Logo</a></h1>
        <HamburgerIcon 
            open={boolean("Open Menu", false)}
          />
        </Navbar>
    </Story>
</Preview>
