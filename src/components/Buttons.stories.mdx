# Buttons

Buttons are used to trigger actions within an application.

<br/>

## Usage

Buttons are used to trigger internal actions within your web applications.

<br/>

## Primary Buttons

Primary buttons are used as a call to action and indicate the most important action on a page.

<Preview withToolbar>
  <Story name="primary">
    <PrimaryButton disabled={boolean("Disabled", false)} onClick={action('button-click')}>Hello world</PrimaryButton>
  </Story>
</Preview>

<br/>

## Secondary Buttons

Secondary buttons are used in this app as a ghost button. They indicate a slight difference in contrast from the primary buttons.

<Preview withToolbar>
<Story name="secondary">
  <SecondaryButton>Hello world</SecondaryButton>
</Story>
</Preview>

<br/>

## Tertiary Buttons

Tertiary Buttons utilize the tertiary color of Aiden's color palette. It's used for the navigation links in the Header components.
